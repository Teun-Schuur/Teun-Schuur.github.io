CPU project

8 bit computer met logic gates


Dit project ben ik volgens mij in 2018 begonnen. De aanleiding tot dit project is dat ik erg gefascineerd was bij hoe computers werken. Vlak daarvoor was ik nog bezig om simpele audio versterkers te maken dat bleek later veel te hoog gegrepen voor een project. Daarom dat ik de digitale kant op ging. Ik begon met een 100 transistors kopen vanuit China om simpele logic gates mee te maken. Toen ik er uiteindelijk een D flip-flop er mee gemaakt had begon het saai te worden. Die week nog kwam ik de YouTube series van Ben Eater tegen. Hij legt uit hoe je een 8 bit computer uit logic gates kan maken. Zijn design was gebaseerd op de SAP1 (Simple As Posible 1). Het komt er op neer dat je dat nog maar net een computer kan noemen. Nadat ik die volledige series had gekeken ben ik gaan ontwerpen. En uiteindelijk voor 80 euro aan IC's gekocht. De computer die ik had bedacht had in plaats van 4 bit address buss 8 bit. dus 256 opcodes, Net genoeg om prime nummers te berekenen! Na ongelofelijk veel tijd en moeite was het eindelijk af. Het had 20 verschillende opcodes, 256 byte ram die je met de raspberry pi kan programmeren en basic IO (input-output) controll. Het slurpte rond de 2 Ampères!!! Totale tijd gekost: ongeveer 300 uur.





Z80

En meteen door naar het andere digitale project! Dit keer vond ik dat ik genoeg wist van hoe een processor in elkaar zat. I wilde er dit keer iets mee doen. En dus was ik op onderzoek gegaan naar een processor die dicht bij de hardware ligt. Dan denk je natuurlijk aan een van de super handige ATMEGA chips of wat dacht je van een STM32, nog veel krachtiger... Ik vond dat toch erg  laag gegrepen. Dan hoef je alleen maar in je favoriete programmeertaal de microcontroller te programmeren. Wat ik wilde is de volgende stap. Niet een paar stappen overslaan en gewoon naar de nieuwe technologie gaan. Wat ik wilde is in Asambly programmeren. Het liefst een basic OS er voor maken. En vervolgens een game er op maken.  Uiteindelijk ben ik op de Z80 processor uitgekomen, een processor uit de jaren 70. De verouderde processor die ook in de grafische rekenmachine zit. De stappenplan ziet er dan zo uit: Wat zijn je eisen Architectuur van de processor begrijpen timing begrijpen Geheugen toevoegen IO toevoegen Beetje aanklooien met code :) Video toevoegen (hier ben ik nu) enz Dus een project die nog gaande is. Tot nu toe erg veel geleerd, ongeveer 200 uur aan besteed. Uiteindelijk heb ik het 2x opnieuw gedaan, telkens weer een stuk veder. Foto links is de eerste versie, rechts is de 2de.  De eerste versie heeft alleen als IO een paar buttons en een Lcd-scherm. De tweede versie heeft op dit moment UART-communicatie.
